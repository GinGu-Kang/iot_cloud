# # 시작일 1.2일 


처음 회원 가입시 
회사 관리자 계정 하나를 제공

회사 관리자만 아이디를 생성할 수 있다.

생성하면 이메일로 이메일 인증 체크

# 인증
* 로그인
* 회사등록
* 비번찾기
* 아이디찾기
* 개인정보수정
* 사원아이디생성
* 이메일인증

회사등록  -> 회사이름 입력 , 대표 이메일 입력 (이메일 인증) , 회사주소 입력  -> 회사코드 대표아이디 생성 
사용자 회원가입 -> 회사코드 입력 유효성 -> 이메일 인증 -> 가입 후 승인대기 
승인 대기 회원 -> 대기중인 회원 목록 보여주기 -> 승인,삭제
회원 권한 부여 -> 사내 회원 목록  -> 권한 부여 -> 팝업창으로 권한 부여,회수
회원 관리 로그 -> 삭제 부여 및 회수 로그 ->
